# streamable-viewbot
Increase your views on Streamable

1. Install python3+ (https://www.python.org/downloads/) with "ADD TO PATH" mark in installer.
2. Requirements:
websocket-client

Steps:
- Launch streamable.py
- Enter link of streamable video
- Enter count of threads
- Enjoy!
